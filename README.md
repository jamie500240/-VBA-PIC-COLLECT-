# 影像自動化歸檔中樞系統
**Image Workflow Engine**

一個讓不擅長整理檔案的人，也能一鍵完成圖片整理、排版、備份與輸出的 Excel 工具。

---

## 一句話版本

一個為「容易誤操作的人」設計的圖片整理工具，同時也是我與 AI 協作開發的實驗紀錄。

---

## 這個專案在做什麼？

如果你也有這些情況：

- 圖片越存越亂
- 檔案名稱隨意、資料夾混雜
- 不小心刪錯就很難救回
- 想整理，但最後總是放棄

這個工具的目標很單純：
只要按幾個按鈕，就能完成匯入、整理、排版、輸出，甚至包含救援機制。
不需要記流程，也不需要理解內部細節。

---

## 為什麼會有這個專案？

起點很日常——我養了一隻貓，也很愛拍我哥的貓、路上的貓。
加上畫圖、刺繡、羊毛氈的過程記錄，還有長年蒐集的各類謎因。
圖片越來越多，整理越來越痛苦，找不到適合的工具，所以決定自己做一個。

在實際使用後，我慢慢發現一件更現實的事情：
我其實不是一個很「會照規則操作」的人。

我會：
- 忘記自己剛剛改了什麼
- 重複複製或刪錯檔案
- 在流程中途改變做法
- 沒有完整理解就開始操作

而更重要的是：
這些情況不只會發生在「使用者」，也會發生在「未來的我自己」。

所以這個系統後來的設計目標變成：
就算使用方式很隨意，也不容易壞；就算壞了，也盡量能救回來。

---

## 讓未來的我好維護

除了避免誤操作之外，這個專案也刻意設計成讓未來的我可以更容易修改。

因為我知道自己有一個習慣：
當系統變大之後，我可能會忘記當初為什麼這樣設計，也可能不敢再動它。

所以這個系統被刻意拆成比較好維護的方式：
- 功能拆成小單位
- 重要設定集中管理
- 每個部分盡量可以獨立理解
- 用清楚註解降低理解成本

目的不是讓它看起來複雜，而是：
未來的我，就算隔很久回來，也還能改得動。

---

## AI 在這個專案中的角色

這個專案不是完全手寫完成的，而是透過 AI 協作逐步完成：

- 我負責決定「我要什麼效果」
- AI 協助生成與修改程式碼
- 我負責實際測試與修正
- 透過反覆修改與重構讓系統逐漸穩定

我不懂程式碼，但我知道我要什麼。
這個過程本身，也是一個實驗——
**一個完全不懂程式語言的人，能用 AI 做出什麼？**

## 設計理念

這個系統不是為了「理想中的使用者」設計的，
而是為了「真實會發生的情況」設計的，包括：

- 會忘記的人
- 會亂點的人
- 會誤刪的人
- 會中途改想法的人
- 以及未來的我自己

設計原則：
- 操作簡單（一鍵完成）
- 錯誤不容易變災難
- 設定集中
- 功能拆小
- 優先考慮能不能救回來
- 也考慮未來能不能修改

## 未來展望

下一個目標是讓註解和說明更清楚，
讓任何人拿到這個檔案，不需要懂 VBA，
也能看懂每段程式碼在做什麼、為什麼這樣寫，
然後自己維護、自己改。

程式碼是工具，不是門檻。

---

## 主要功能

- 一鍵匯入與圖片去重（MD5 指紋比對）
- 自動瀑布流排版，支援三種排版模式
- 圖片目錄與索引系統，支援新舊目錄雙向超連結
- 遺失檔案多軌救援機制
- PDF 報告輸出（4／6／8 宮格圖片報告 + 橫式目錄報告）

---

## 環境需求

- Windows 作業系統（不支援 Mac）
- Microsoft Excel，建議 64 位元版本
- 需啟用巨集

---

## 如何使用

1. 下載 `.xlsm` 檔案
2. 開啟後啟用巨集
3. 在「操作介面」設定專案名稱與參數
4. 點擊「開始作業」匯入圖片
5. 點擊「全面排版」整理圖片工作表
6. 視需要使用「匯出圖片」或「匯出目錄」產生 PDF 報告
